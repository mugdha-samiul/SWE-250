# PriceSpy

PriceSpy is an Android application designed to help you explore different online shops for a product and choose the most convenient option for yourself. 

## Features

- **Browse Product Categories**: Explore various product categories such as meat & fish, personal care, etc.
- **View Product Information**: Access the information about the same product in different online shops. 

## Technologies Used

- **Java**: Programming language used for developing the application.
- **Jsoup**: Java library for web scraping, used to fetch product details from websites of different online shops.
- **Android Studio**: Integrated Development Environment (IDE) used for Android application development.
